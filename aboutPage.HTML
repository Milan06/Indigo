About the Website

This webpage was created by a bunch of high school students who are eager to learn more code! Over the past few weeks we have taken a coding bootcamp that specializes in Front-End Web Development.We have learned about HTML5, CSS3, JavaScript, and so much more. Our website features a variety of games that utilize all of the different languages we have learned. We hope you enjoy the games we have created!

The Languages That We Have Learned!

The main languages we discussed over the last few weeks were HTML5, CSS3, and JavaScript.


The first language we discussed was HTML5. HTML stands for Hyper-Text Markup Language and it is primarily used to create webpages that can be seen through the world-wide web.

The final language we discussed through the class was JavaScript. JavaScript is used to make webpages interactive. We have used JavaScript in most of our games which allowed us to make them interactive.

The second language we learned during this bootcamp was CSS3. CSS stands for Cascading Style Sheets and it's used to make webpages look more organized, neat, and pretty.
